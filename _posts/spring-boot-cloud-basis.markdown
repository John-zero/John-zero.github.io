## 三者关系

Spring --> Spring Boot --> Spring Cloud

Spring 是核心, 是基础. 

Spring Boot 用以简化 Spring 搭建和开发过程. (脚手架, 模板)

Spring Cloud 基于 Spring Boot 为微服务体系架构开发提供一整套解决方案.
